# 5주차 실습 기록

## 선택한 템플릿 및 주제
- space_shooter.py

## 사용한 AI 프롬포트
- 3주 안에 만들고자 하는 구현이 가능한지 물어봄.
- 코드를 추가할 때마다 전체적인 코드 정리를 부탁함.
- 해당 코드에 어떤 아이디어를 더 추가하면 좋을지 물어봄.
- 오류가 생길 때마다 수정해달라 부탁함.

## AI 답변에서 도움이 된 것
- 코드를 만드는 걸 자체를 못하다보니까 코드를 만들어주는 것 자체가 도움이 된 것 같다. 특히 보스전 코드를 제작 요청할 때 "보스전 코드 오류 없이 가능한가?" 라고 단순하게 물었는데도 보스 전용 체력바와 보스를 처치했을 때 나오는 드랍 아이템 등을 코드로 만들어서 준 것이 많은 도움이 되었다.

## AI 답변을 수정하거나 버린 것
- 원래 있던 코드 자체가 갑자기 수정되거나 삭제되는 경우가 있어서 다시 코드를 되돌려달라고 여러번 요청했다. 웬만하면 버린 것은 없는 거 같다.
- 화면 위에 레벨1 이라고 떴던 글씨가 사라진 오류 수정, HP가 없는데도 game over 화면 없이 초기화되는 오류 수정 등.
