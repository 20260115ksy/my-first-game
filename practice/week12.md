# 12주차 실습 기록

## 오늘 실행해본 코드 중 가장 흥미로웠던 것
- BSP 던전 생성이 가장 흥미로웠다. 공간이 분할되는 과정과 겹치는 길을 삭제하는 과정을 쉽고 빠르게 잘 볼 수 있어서 신기했다.

## 파라미터를 바꿨을 때 어떤 변화가 있었는지
- 구조, 모양, 크기, 생성 속도 등이 눈에 띄게 달라졌다.  

## 기말 프로젝트에서 프로시저럴 생성을 활용할 수 있을지
- 수작업으로 오랜 시간이 걸릴 정도로 크게 만들 것 같지 않아서 프로시저럴 생성을 기말 프로젝트에서는 쓰지 않을 것 같다. 하지만 앞으로 만들게 될 게임에는 효율적이고 유용하게 사용 가능할 것 같다.
